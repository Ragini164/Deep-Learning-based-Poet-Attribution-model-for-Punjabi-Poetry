16. Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke
Main Vaisan Na Rahisaan Horhe, Kaun Koi Main Jaandi Nu Morhe,
Mainu Murna Hoeya Muhall, Sir Te Mihna Cha Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Jogi Nahi Eh Dil Da Mita, Bhul Gayi Main Pyaar Kyun Kita,
Mainu Rahi Na Kuch Sambhal, Us De Darshan Pa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Es Jogi Mainu Kahiyaan Laaiyaan, Hein Kaleje Kundiyan Paaiyaan,
Ishq Da Paaya Jaal, Mitthi Baat Suna Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Main Jogi Nu Khoob Pachhata, Lokan Mainu Kamli Jaata,
Lutti Jhang Siyaal, Kanni Mundraan Pa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Je Jogi Ghar Aave Mere, Muk Javan Sab Jhagre Jhedre,
La Seene De Naal, Lakh Lakh Shagan Manaa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Maaye Ni Ik Jogi Aaya, Dar Saade Us Dhuan Paaya,
Mangda Heer Siyaal, Baitha Bhes Vata Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Taane Na De Fufi Taai, Aithe Jogi Nu Kismat Lai,
Hun Hoeya Fazal Kamal, Aaya Hai Joga Sidha Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Maahi Nahi Koi Noor Ilahi, Anhad Di Jis Murli Vaahi,
Muthiyo So Heer Siyaal, Dhadhe Kaaman Pa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Lakhaan Gaye Hazaaran Aaye, Us De Bhait Kise Na Paaye,
Gallan Taan Moosay Naal, Par Koh Toor Chadha Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Abdahoo Rasool Kahaaya, Vich Mi'raj Buraq Mangaaya,
Jibraeel Pakar Lai Aaya, Haraan Mangal Ga Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Es Jogi De Suno Akhaade, Hasn Husain Nabi De Pyaare,
Maario So Vich Jaddaal, Paani Bin Tarsa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Es Jogi Di Suno Kahaani, Sohni Dubbi Doohe Paani,
Phir Rallia Mahiwaal, Saara Rakhit Lutta Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Daanvaan Doli Lai Chale Kherhe, Muddh Kadeemi Dushman Jehre,
Raanjha Taan Hoeya Naal, Sir 'Te Tamak Dhara Saja Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Jogi Nahi Koi Dooja Saaiya, Bhar Bhar Piala Zauk Pilaaya,
Main Pee Pee Hoee Nihaal, Ang Bhbhoot Ramaa Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Jogi Naal Karendey Jhere, Kahe Pa Baithe Kaazi Jhere (Bherhe),
Vich Kaidon Paai Mukal, Kooda Bara Laga Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.

Bullah Main Jogi Naal Vyaahi, Lokan Kamliyan Khabar Na Kaai,
Main Jogi Da Maal, Panje Peer Mana Ke,
Main Vaisan Jogi De Naal, Mathhe Tilak Laga Ke.
