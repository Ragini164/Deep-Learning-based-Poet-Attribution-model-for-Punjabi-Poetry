Khedde khirke aakhda hai:

Dharti godi banaai
Main is vich kheddiya,
Mitti jadhaan gadaai
Main bhojan khichhiya,
Raj raj peeta neer
Main aado vageediya,
Hoya main namgeer
Te trailan chakhiyaan,
Badal arshon sad
Main paani khichhiya,
Nhaata main raj raj
Te nha nha nikhraya,
Sooraj paaso dhuh
Main kirnaan leetiyaan,
Niggi kiiti rooh
Main vadhiya mauliya,
Channan te nigh kad
Main leeta dhupp to,
Laayi aap vich gad
O maano dhupp main,
Taann chaandani shaal
Main sutta raat nu,
Dalk tarriyaan naal
Main laiyaan loriaan,
Nhera liya bithaal
Main pehiredaar si,
Koi na dei uthaal
Main raatein suttiyaan;
Jo kuch mere haal
Si mainu sajiya,
Aape vich bithaal
Main aapa paaliya,
Pal pal rang jamai
Main bhariya dodiyaan,
Dodiyaan khedde layai
Main saara khir piya,-
Mahkaya mushk machai
Te laptaan chhaddiyaan,

Haan, hun jad lapat buhaaye
Main dulh dulh pai rihha,-
Kheddiyaan naala bharaan
Main jholiyaan adiyan,
Anhonda aap vandaye
Main vandia ja rihha.

'Dena' banna da roop hai 'khedde khir piaan',
'Dena' dena rang anoop hai chhad da mushkiaan.
